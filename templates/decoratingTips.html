  • 施工验收

    3大“须知”、水电泥瓦验收等

    6大‘陷阱’:
  1. 验房收房时，开发商必须提供‘三书两证一表’。

    “三书”指《住宅质量保证书》、 《住宅使用说明书》、《建筑工程质量认证书》

    “两证”指《商品房竣工验收合格证》、《新 建住宅商品房准许交付使用证》

    “一表”指《竣工验收备案表》。只有以上证件齐全，才能说明商品房质量符合国家认可标准。

  2. 房屋的实际面积与合同面积误差在3%之内属于正常范围

    房屋实测面积小于合同约定面积且误差比在3%以内(含3%)部分的房价款及利息由出卖人返还买受人，面积误差比超出3%的部分房价款由出卖人双倍返还买受人。

    计算公式:面积误差比= (实测面积-预测面积) +预.测面积x100%。

  3. 层高的高度一般在2.8米

    如果层高未达到要求，一般按购房合同中的约定要求赔偿。一般购房合同约定层高缩水按比例赔偿。

    计算公式:赔偿额=层高缩水比例x房价款。

  4. 专业验房机构可以出具专业验房报告,比自己验房更具有安全保障。
  5. 除了自家室内,还需检测公共区域和楼下卫生间的防水。
  6. 验房的时间分好坏,雨天验房优于晴天验房。
    墙顶地验收:
  1. 墙面裂缝宽度>0.3mm、空鼓面积>400cm均需整改。
  2. 用阴阳角尺或红外线检测仪检测阴阳角,误差小于5mm才可以正常验收。
  3. 用红外水准仪、卷尺检查地面(*厨房/卫生间除外)的平整度误差应小于2cm。
  4. 地面流向地漏处的坡度要控制在2%--5%之间。
    水电气验收:
  1. 强电箱内部回路应有配置标识、漏电保护装置。
  2. 燃气表、燃气报警器应能正常使用，若不能必须由燃气公司_上门调整。
  3. 确认厨卫吊顶处需预留可正常使用的通风口。
    二手房验收:
  1. 除了拆改部分，二手房直接按新房验收标准进行检查。
  2. 有“私拆乱建”等行为的二手房不可直接验收。
  3. 二手房账户缴费需留存过户，新购房业主有权要求原业主补缴欠款费用。
  4. 防水渗漏需维修的话要找邻居家配合做防水测试。
    5大"须知":
  1. 装修预算=设计费+人工费+材料费(主材和辅材)+家具软饰+家电设备
  2. 先定风格再选材料，材料决定整体预算。

    日式、北欧、现代简约风，造型少，风格简单更加省钱，装修费用在800-1000元/平方米; 美式、中式、欧式涉及到的装饰造型多，整体造价更高，装修费用在1500-2000元/平方米。

  3. 合理安排装修项目,要留出20%的预算应对必要增项。
  4. 二手房装修比新房装修预算高20%--30%左右。
  5. “大型家庭设备”除主机费用外牵涉施工安装等-系列费用，需提前规划整体预算。
    省钱要点:
  1. 非常小的面积误差也会影响各项费用计算。

    面积哪怕只有几平米误差，都会对各项费用产生影响，从而导致预算的增加。

    ①套内面积:决定设计费、水电基础费。

    ②地面墙面面积:决定瓷砖和木地板面积。

    ③其他面积:铝合金、踢脚线、窗帘、窗台石、淋浴房等。

  2. 技术含量高的活如设计费、水木瓦油施工费不能省;技术含量低的活如敲墙，运送垃圾费可以省。
  3. 通过控制“瓷砖、地板、涂料”这三类材料的耗损度能有效节省预算。
  4. 购买建材家具考虑代工厂品牌或者工厂店，预算能省10%-20%。
  5. 软装饰品建议购买网购，预算能节省60%。
  6. 装好的房子不必请专业公司除甲醛，能节省3000 ~ 5000元。
    5大"陷阱":
  1. 签合同交订金时要明确写的是“订金”，订金可退，定金不可退。
  2. 合同中的施工项目要与施工图上的保持一致。
  3. 合同中的材料明细需写明材料种类、品牌、型号、规格、颜色、用量。
  4. 合同中需注明:装修付款方式、时间、比例、次数、工程进度，缺一不可。
  5. 合同注明增项金额不得超总预算5%-15%，否则由施工方承担。
    签订要点:
  1. 除“工程材料费”外，还需确认合同中其他费用(*管理费/设计费/税费)的收取金额、比例、服务明细。
  2. 在合同中明确“监督条款”，监理需2- -3天到场- -次设计师需5-7天到场一次。
  3. 检查并明确合同中是否标注明确需单独收费的施工工艺(*地砖上墙、非常规贴砖法等)。
  4. 施工前，公共区域需做好保护，避免材料进场/垃圾清运时对墙、地面造成损坏。
  5. 检查并明确合同中是否包含“易漏”项目(*垃圾清运、刷乳胶漆、石膏/抹灰找平等)。
  6. 合同中应注明“超时交房违约金”明细(*包括赔付金额、计算单位、计日方式)。
  7. 合同中需注明施工验收的“空气质量”符合国家标准，否则需返工。
  8. 检查并明确合同中“售后服务”明细(*包括售后时长、可售后服务项目)。
    装修公司VS设计师:
  1. 选择对施 工工期有明确限制的装修公司。
  2. 一般独立设计师比装修公司设计师更专业。
  3. 找施工队装修建议请监理监工。
    全包VS半包:
  1. 无论选择哪种装修模式，都要考察“在建工地”。
  2. “全包”模式的工期为30-60天;半包,清包则为60-90天(90m为例)。
  3. 不同装修模式缴费比例不同，全包多为70%,25%,5%;半包、清包多为55%,45%,5%。
  4. 从售后体验上来说，半包,全包要比清包好。
  5. 半包、清包需业主自行去建材市场采购建材,全包不用。
  6. “全包”模式下非通用型的施工项目如背景墙、吊顶设计等都需要单独收费。
  7. 留意全包模式下“全包”但实际容易造成单独计费的 项目(*主材 替换、非标配项目、材料限制等)。
  8. “半包”模式下业主买主材，装修公司提供辅材。
    业主手续:
  1. 装修不得擅自开工，需提前去物业办理开工手续，交纳装修押金和垃圾清运费。
  2. 办理装修手续时，需要向物业问清楚建渣堆放点。
  3. 拆改(*包括承重墙、飘窗、阳台等)必须经由物业许可。
  4. 施工交底中所有项目均需达成书面共识，切忌口头协议。
    开工交底:
  1. 开工前设计师必须将交底材料准备齐全带至施工现场。
  2. 施工方需提前准备好施工进度表、节点验收表。
  3. 施工交底时，业主、设计师、监理、工长需同时到场，缺一不可。
  4. 施工前，公共区域需做好保护，避免材料进场/垃圾清运时对墙、地面造成损坏。
  5. 施工方不能擅自拆改煤气、天然气管道、旧暖气片、暖气管道等。
  6. 水电交底前需再次确认水电施工等特殊电器点位。
  7. 水电交底现场仍可提出改动需求，如开关、位置等。
    5大“须知”:
  1. 优先选择经过“环境质量体系”认证的环保材料。
  2. 主材辅材的选购时间根据施工进度来定。
  3. 为避免后续维护难题,辅材更需要选择有保障的大品牌。
  4. 适合网购的建材有:厨卫五金、开光插座、油漆墙纸、灯具等。 不适合网购的建材有:水电材料、瓷砖、地板、定制家具、门窗等。
  5. 网购建材在验货时要清点产品包装、型号、颜色、尺寸、数量，缺一不可。
    主材选购:
  1. 为避免多买浪费,壁纸和涂料用量可以按公式估算。
  2. 购买瓷砖、地板时要比实际铺设面积多算5%-10%的量,避免耗损率。
  3. 选购瓷砖不一定非要选大品牌。
  4. 实木复合地板性价比最高。
  5. 五金件质量稳定、退换方便，建议网购有质量保证的大品牌产品即可。
  6. "室内门多选木门，其中实木门隔音效果最好，实木复合门性价比最高。
  7. 考虑隔音隔热，窗户建议选断桥铝窗框+双层夹胶玻璃。
    辅材选购:
  1. 水电材料-定要有 合格的质检报告，优先选择有品牌知名度的产品。
  2. 不同区域选择不同特性的防水材料，优先选择质量售后有保障的大品牌。
  3. PPR管是性价比最高的水管。
  4. 电线要选用纯铜芯线缆，家庭最常用的电线型号是1.5方和2.5方。
    3大"陷阱":
  1. 装修小白、业主时间不充裕的情况下需要请监理。
  2. 第三方专业监理通常比装修公司的免费监理更能保证工程质量。
  3. 水电改造及主材安装阶段业主必须亲自在场，其余环节不需要。
    拆改监工:
  1. 承重墙绝对不能拆除。
  2. 集中供暖的房子，改造暖气的时间最好是在非供暖季。
  3. 燃气管道要装在墙外且避免使用木质等易燃材料。
  4. 顶面的横梁不能拆。
    水电泥瓦监工:
  1. 水电铺设的线路和数量要和施工图纸-致。
  2. 水管电线的品牌、型号、厚度和直径要和报价单致。
  3. 施工前要对“地漏”和“下水口”进行封闭保护。
  4. 墙地面同时铺设瓷砖时，需保证“墙砖压地砖”。
  5. 地暖布管采用“螺旋形”的铺设方式，取暖更均匀。
    木工油漆验收:
  1. 石膏板吊顶，转角处石膏板要裁成“7字型”安装。
  2. 木工工程中出现0.2mm以上明显缝隙需重新返工。
  3. 踢脚线在墙角处相交的地方,边缘要进行45度角裁切。
  4. 吊顶木龙骨安装前要整面涂刷防火涂料。
  5. 避免油漆发白，不要在下了大雨后喷涂油漆。
    主材安装监工:
  1. 铺贴地板的最佳室内温度为16°C -30°C。
  2. 地板铺贴要在基本装修施工完成后再进行。
  3. 地板铺贴完48小时内，必须用纸盖上，且不要在地板上频繁走动和放置重物。
  4. 安装水龙头前要放水冲洗水管 。
  5. 马桶排污口要密封。
    3大“须知”:
  1. 施工验收要准备卷尺、响鼓垂、墙体探测仪、直角尺、靠尺、塞尺、水平仪、验电器9种工具。
  2. 施工项目需逐一验收，不能最后统一验收。
  3. 验收合格后，施工方需开具“家装工程保修单”。
    水电泥瓦验收:
  1. 关闭总闸验收，水电表不能有转动的情况。
  2. 水电工程至少要进行两次验收。
  3. 水路打压试验，试验后的压力值不能超0.05MPa。
  4. 查验水电路铺设情况，要确认两者交叉时“电路在.上水路在下”。
  5. 验收冷热水管、强弱电线布线时主要查看“间距”。
  6. 验收墙地面的瓷砖空鼓面积不能大于400cm。
  7. 验收地面平整度的落差不能超过3mm。
    木工油漆验收:
  1. 检验门窗套和墙面的缝隙不能超过5mm。
  2. 踢脚线钻孔的间隔应在40cm左右。
  3. 地板间隙不超过1mm。
  4. 涂漆后的墙面不是越光滑越好。
  5. 手摸墙面有白粉涂料是正常的。
  6. 木器漆工程中易漏刷家具侧面、背面、底面及柜体内，需重点检查。
    主材安装验收:
  1. 空调孔要封堵。
  2. 橱柜的洗菜盆下方柜体要贴防水铝箔纸。
